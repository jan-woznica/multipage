APIs

Developři napsali i další funkcionality jako:
 - Browser Application Programming interfaces (APIs) - tvorba HTML a CSS, práce s videem atd...
 - Third-party APIs - umožňují vložení funkcionalit z jiných webových stránek
 - Third-party frameworks a libraries - aplikovatelné na HTML pro zrychlení tvorby stránek
 
Javascript může: uchovávat hodnoty, operovat s kousky textu znamých jako stringy, spustit kód na základě určité události a mnoho jiného
 Mnohem více vzrušující je funkcionalita navrh javascript jazyka - Application programming interfaces (APIs) - poskytují extra možnosti k použití v javascriptu
 API jsou bloky kódu, které umožňují developerům implementaci programů, které by jinak byly náročné k implementaci

 API lze rozdělit na dvě kategorie: 
 Browser APIs - vestavěny do webového prohlížeče a jsou schopny odkrýt data z okolí PC prostředí nebo:
  - DOM (Document Object Model) API: manipulace s HTML, CSS
  - Geolokační API: získání geografické polohy
  - Canvas a WebGL API:vytvořit 2D a 3D grafiky
  - Audio a Video APIs: práce s multimédii
 Third part\ APIs - nejsou vestavěny v prohlížeči defaultně, většinou je musím získat někde na webu
 - Twitter API: umožní zobrazení mých posledních tweetů na mém stránce
 - Google Maps API, OpenStreetMap Api: vložení obyčejných map do stránky a dalších funkcionalit

Co dělá Javascript na mém stránce

Jak již víme z předešlých kurzů, tak při načtení stránky spuštím svůj kód: html, css a javascript uvnitř exekučního prostředí. Javascript se často v tomto případě využívá k modifikaci HTML a CSS pomocí APIs

Variables - proměnné

Proměnné jsou kontejnery, které ochovávají hodnoty
proměnné je důležité pro vytvoření čehokoliv zajímavého v programování

Začínáme tím, že nastavíme proměnnou pomocí let fráze, následované pomocí názvem proměnné, odstředník na konci označuje, že tvrzení končí
 let myvariable;
 Po pojmenování proměnné ji můžeme dát hodnotu let myvariable = "Bob";
 Proměnnou vyvolám tím, že ji zavolám: myvariable;
 Přiřazenou hodnotu mohu později v kódu změnit myvariable = "Radek"
 Pozor! proměnné mohou mít i jiné hodnoty:
 string - "text uvnitř uvozovek bez významu"
 number - číslovky 10, 20 atd..
 Boolean - True / False slova
 Array - struktura umožňující uchování několika hodnot v jedné zmínce [1, "Bob", "Radek"]
 Object - Cokoliv v javascriptu může být objekt a může být ochováno v proměnné

Operátory

Operátor je matematický symbol, který produkuje výsledek na základě dvou hodnot
Existuje mnoho operátorů, ale následují jenom ty nejzákladnější

Sčítání - sečte dvě hodnoty dohromady - + - 6 + 9 nebo "HEllo" + "world"
 odečítání, násobení, dělení - to, co v normální matice - -*/ - 
 rovná se - přiřadí hodnotu k proměnné - = - let myvariable = "Bpb" 
 Přísná rovnost - provádí test, zda jsou si dvě hodnoty rovné a jsou stejného datatypu a vrátí true/false - === - mojevari === 4
 Not, Does-not-equal - vrátí opačnou hodnotu, než normálně. Vedle rovná se kontroluje, jestli si hodnoty nejsou rovny - !, !== - Základní hodnota je true, ale not vrátí false kvůli negaci - let var = 3, !(var === 3) - false

Conditionals - Podmínky

Jsou struktury kódu, pro kontrolu zda je hodnota true nebo false<
Velmi běžná forma podmínek je ic...else 

 let myvar = "chovolate";
 if (myvar === "chocolate") { 
   alert("Miluji čokoládovou zmrzku");}
 else {
   alert("Škoda, čokoládová je moje nej");}
